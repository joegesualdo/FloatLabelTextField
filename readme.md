## JPGInput
> A Fancy iOS input

## Install
```ruby
# Podfile
pod 'JPGInput', '~> 0.1.5'
```

## Usage
```Swift
...
let firstNameInput = JPGInput(label: "First name", placeholder: "First name")
view.addSubview(firstNameInput)
...
```
